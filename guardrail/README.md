# guardrail — 修 CLAUDE.md 的結構病

CLAUDE.md 寫久了容易長胖：重複的規則、從來沒被觸發的 section、為了以防萬一加的條目。guardrail 就是幫你刮骨療毒的工具。

它先問一個問題：這段東西真的需要存在嗎？能刪的就刪，能濃縮成一行就不留三段話。找不到可刪的才考慮修飾。

診斷分兩層：先掃結構性的問題（dead section、重複規則、過度描述），再檢查格式（行數、缺 section、模糊規則）。發現問題會追根究柎——不是只說「太長了」，而是告訴你為什麼會變這麼長，怎麼修最省事。

修完會報告 before/after：刪了多少行、殺了幾個 dead section、讀起來順不順。
